Data Currency & Sources

Data Sources

The data in this dashboard are derived & refreshed from FEMA's Disaster Declaration Summaries. These data are refreshed every 20 minutes and are available to download by anyone. Links to the data in various formats, and more information, can be found at the FEMA's Disaster Declarations Summaries page (below).

Note that, while most data fields and attributes are retained by this dashboard, the geospatial data is (mostly) aggregated to the level of the FEMA Declaration String field (often spanning multiple counties/tribal areas), whereas the tabular Disaster Declaration data themselves are (usually) county-level.

FEMA Disaster Declaration Summaries

Data Currency

This dashboard is updated once daily through an automated process that connects directly to FEMA's Disaster Declarations Summaries. Therefore, allow up to 24 hours for new Disaster Declarations to be reflected in this dashboard.

GIS Developers: This dashboard is updated using an ArcGIS Jupyter Notebook in ArcGIS Online. A Scheduled Task runs the notebook on a daily schedule, with a webhook configured to send an email on script completion detailing whether new features were added.
