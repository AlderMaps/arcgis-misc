Help & How-To

Reset the Dashboard

To remove ALL filters and reset the dashboard (i.e. zoom out and show all counties), select the circular arrow in the lower right:

Filter the Data

Use the filters on the left to view Declarations for a particular e.g. fiscal year or state. You may also set additional filters for Incident Type, Declaration Title and FEMA Declaration String.

The header includes two additional filters, for COVID-19 and Entities. By default, only non-COVID-19 declarations are displayed, but you may alternatively view only COVID-19 Declarations, or all Declarations.

Disaster Declarations may be issued at the state (or equivalent) level, county (or equivalent) level, or for the various types of tribal entities (American Indian, Alaska Native, Native Hawaiian). Use the Entities filter to view these categories separately or together.

Use the Incident Type Pie Chart

The Incident Type pie chart (lower right) shows the counts and relative proportions of the various Declaration types (e.g. Fire, Flood, Hurricane) present in the current map view.

The chart is dynamic and interactive; when the data is filtered to a given state and/or year, the chart will update to reflect only those data. Conversely, the map data and list can be further filtered by selecting individual incident types/slices of the chart.

View Declaration Details

To view the details (e.g. start date, end date) of a given Disaster Declaration:

1.Use the filters in the left sidebar and/or zoom the map to narrow the number of Declarations displayed in the map.

2.Select a single shape in the map, OR select a declaration from the list in the upper right.

3.Select the "Details" tab (below the pie chart, lower right) and scroll to view the full Declaration details.

These details match (and are derived directly from) FEMA's Disaster Declarations Summaries, which are available for download in various data formats. See the "Data" tab for more information.
