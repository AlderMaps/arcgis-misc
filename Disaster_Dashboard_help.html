Help & How-To

Reset the Dashboard

Filter the Data

Use the filters on the left to view Declarations for a particular e.g. fiscal year or state. You may also set additional filters for Incident Type, Declaration Title and FEMA Declaration String.

The header includes two additional filters, for COVID-19 and Entities. By default, only non-COVID-19 declarations are displayed, but you may alternatively view only COVID-19 declarations, or all Declarations.

Disaster Declarations may be issued at the state (or equivalent) level, county (or equivalent) level, or for the various types of tribal entities (American Indian, Alaska Native, Native Hawaiian). Use the Entities filter to view these categories separately or together.

View Declaration Details

To view the details (e.g. start date, end date) of a given Disaster Declaration:

◗Use the filters in the left sidebar and/or zoom the map to narrow the number of Declarations displayed in the map.

◗Select a single shape in the map, OR select a declaration from the list in the upper right.

◗Select the "Details" tab (below the pie chart, lower right) and scroll to view the full Declaration details.

These details match (and are derived directly from) FEMA's Disaster Declarations Summaries, which are available for download in various data formats. See the Data Currency and Sources tab for more information.
