About the Declarations

This dashboard allows users to view the geographic extent of FEMA Disaster Declaration from 2013 to the present. Disaster Declaration are usually county-based and encompass one or more counties, but may also be state-wide or defined by tribal areas. In this map, Disaster Declarations are aggregated into a single shape per Disaster Declaration string (e.g. "FM-5608-WY").

This dashboard also provides a summary of the details of a given Disaster Declaration, available in the "Details" tab in the lower right. These details are derived directly from FEMA's Disaster Declarations Summaries (see the "Data Currency & Sources" tab). To view details for a given Declaration, select a shape in the map or select a row from the list above the "Details" tab.
